About this Website


This website allows you to convert images which contain text (perhaps an essay or something of the sort) into txt files using Tesseract OCR by Google, along with some help from OpenCV. You can then use two txt files in our plagiarism detection area to see what percent of those files are plagiarising off each other. This is done using PySimilar. This project is our submission in the Hack3 hackathon, on June 26, 2021.
